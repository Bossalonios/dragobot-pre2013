Syntax: !dragobot <command>
Each command has its own parameters.
[no command]  Prints the main menu message.
about         Prints an about message.
gamehelp      Prints a help message for the games.
help          Prints this message.
quit          Makes me quit. Can only be used by my operator.
triggers      Lists all triggers that I will respond to.
triggerhelp   Prints a help message explaining what each trigger does.
